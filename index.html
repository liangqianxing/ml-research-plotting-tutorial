02 / PRINCIPLES

先定义比较，再选择图形

一张科研图只承担一个核心论点。颜色、位置和标记都应服务于这个论点。

01

报告分布，而不只报告均值

交叉验证、不同随机种子或独立重复实验应显示标准差、置信区间或原始样本点。

mean ± std
02

使用一致的比较基线

固定数据划分、预处理和评价指标；让图中的差异真正来自模型或实验变量。

same protocol
03

确保灰度与色觉差异下仍可读

同时使用线型、点型和直接标签，不把类别区分完全交给颜色。

color + shape

诊断收敛、过拟合与实验波动

04 / PLOT INDEX

按研究问题选择图型

评价模型时，图型应与问题、数据结构和不确定性来源一一对应。

分类性能ROC / PR

模型能否区分正负样本？

类别不平衡时优先补充 PR 曲线，并报告随机分类器基线。

可信度CALIBRATION

预测概率是否可信？

配合可靠性曲线、样本直方图以及 ECE 或 Brier score。

错误结构CONFUSION

错误集中在哪些类别？

按真实类别归一化，并在单元格中同时给出比例或样本数。

可解释性IMPORTANCE

哪些变量驱动预测？

优先使用置换重要性或 SHAP，并展示重复采样产生的误差。

05 / CODE RECIPES

最小可复现代码

先固定画布、字体和导出参数，再编写具体图形，能显著减少论文排版返工。

06 / EXPORT CHECKLIST

投稿前检查

  • 01尺寸

    按期刊单栏或双栏的最终宽度预览，避免后期缩放导致文字过小。

  • 02格式

    线图优先导出 PDF 或 SVG；像素图至少使用 300 dpi。

  • 03字体

    统一字号与字体家族，嵌入字体并检查数学符号。

  • 04复现

    保存原始数据、绘图脚本、环境版本和随机种子。
