# 尺寸与导出

最终图片必须在论文实际栏宽下可读。尺寸、字号和线宽应该在绘图时确定，而不是在排版软件中反复缩放。

## 从最终栏宽开始

常见目标尺寸仅供初始参考：

| 场景 | 宽度 | 建议基础字号 |
| --- | --- | --- |
| 单栏论文图 | 约 85 mm / 3.35 in | 7–9 pt |
| 双栏论文图 | 约 175 mm / 6.9 in | 7–10 pt |
| 演示文稿 | 16:9 页面内约 10–12 in | 16–24 pt |

先查阅期刊作者指南，再设置 `figsize`。

## 矢量与位图

- 折线、散点、文字和示意图优先使用 PDF 或 SVG；
- 照片、显微图和高密度热图使用 TIFF 或 PNG；
- 位图通常至少 300 dpi，线稿可能要求更高分辨率；
- 检查字体是否嵌入，避免投稿系统替换字体。

```python
fig.savefig("figure-2.pdf", bbox_inches="tight")
fig.savefig("figure-2.png", dpi=300, bbox_inches="tight")
```

## 多面板编号

面板标签应使用统一位置与格式，例如粗体 `(a)`、`(b)`。使用 Figure 坐标放置标签，可避免坐标轴范围变化导致偏移。

```python
for label, ax in zip(["(a)", "(b)", "(c)"], axes.flat):
    ax.text(
        -0.12,
        1.04,
        label,
        transform=ax.transAxes,
        fontsize=9,
        fontweight="bold",
        va="bottom",
    )
```

## 导出后检查

1. 在最终显示尺寸下查看文字和标记；
2. 放大 PDF，确认线条仍是矢量；
3. 转为灰度，检查类别是否仍可区分；
4. 检查图例、轴标签和面板标签是否被裁切；
5. 在另一台设备上打开，确认字体没有替换。

::: warning 不要用截图交付
截图会丢失矢量信息、真实尺寸和色彩管理。最终图片必须从绘图脚本直接导出。
:::
