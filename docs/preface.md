# 00 前言

机器学习论文既要呈现训练损失、验证分数、消融实验等定量证据，也要用方法总览、模型架构和实验流程解释研究思路。真正困难的部分不是把数字或方框放到画布上，而是决定一张图应该支持什么结论、引导怎样的阅读顺序。

本教程围绕一个原则展开：**每张科研图只承担一个清晰论点，同时保留读者判断该论点是否可靠所需的证据。**

## 本教程解决什么问题

- 如何判断应该使用折线图、点图、箱线图还是热图；
- 如何呈现多个随机种子、交叉验证和置信区间；
- 如何画学习曲线、ROC、PR、概率校准和混淆矩阵；
- 如何组织消融实验，避免柱状图制造夸张差异；
- 如何输出适合论文单栏、双栏和演示文稿的图表；
- 如何用 PowerPoint 绘制方法总览、模型架构和实验流程；
- 如何选择科研配色、形状、箭头、字体与视觉层级；
- 如何让 Figure 1 引人注意，同时保持专业、克制和易读；
- 如何保存数据、脚本与环境信息，使图表可以复现。

示意图章节使用 CLIP、BYOL、Glow-TTS 等 NeurIPS/ICML Oral 论文原图做案例。每张图都标明会议、年份、Figure 编号、原始图注和论文链接；教程只分析设计方法，不把自制仿图冒充论文案例。

## 建议学习路径

| 阶段 | 内容 | 结果 |
| --- | --- | --- |
| 基础 | 绘图原则、环境和样式 | 建立项目级视觉规范 |
| 图型 | 学习曲线、分类评估、分布比较 | 根据研究问题选择图型 |
| 实验 | 消融实验和多随机种子 | 正确呈现差异与波动 |
| 交付 | 尺寸、字体、格式和检查表 | 生成可投稿的最终图 |
| 示意图 | PPT 工作流、配色、形状、总览与精修 | 讲清方法并建立统一视觉语言 |

## 示例约定

数据图表示例主要使用 Python、Matplotlib、Seaborn 和 scikit-learn；方法示意图以 PowerPoint 中可复现的基础形状、连接符和参数为主。代码默认数据已经整理为整洁数据格式：每一行是一条观测，每一列是一个变量。

论文原图均从公开论文 PDF 裁取，仅用于教学、评论与来源分析，著作权归原作者及出版方。引用这些图时应保留论文标题、Figure 编号和来源链接；自己的论文应复用设计原则，而不是复制具体图形内容。

```text
seed  model      dataset  split  metric    value
0     baseline   A        test   macro_f1  0.842
1     baseline   A        test   macro_f1  0.851
0     proposed   A        test   macro_f1  0.891
```

::: tip 先保存数据，再画图
不要让绘图脚本重新训练模型。训练过程输出结构化结果文件，绘图脚本只读取结果并生成图片，这能显著降低复现成本。
:::
